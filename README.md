# akintolarahmah.github.io

A simple personal homepage for Akintola Rahmah. Open `index.html` locally to preview the site, or view it deployed via GitHub Pages at `https://akintolarahmah.github.io` after pushing to the `main` branch.

Preview locally:

Run:

```
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

Note: the contact email in `index.html` is currently a placeholder (`akintola@example.com`). Replace it with your real email or preferred contact method before publishing.
